HOW TO USE (3 EASY STEPS):
================================================================================

STEP 1: Find the File
   Location: widget-tables-complete.html
   Size: Should be approximately 530KB

STEP 2: Open It
   Method A (Easiest): Double-click the file
   Method B: Drag file into your web browser
   Method C: Right-click → Open With → Choose your browser

STEP 3: Use It
   - Click any table group (blue cards) to expand
   - Click a specific table to see available depths
   - Click a depth to view the decompression schedule
   - Use "Back" button to navigate

================================================================================

1. Open widget-tables-complete.html
2. You should see a header "Dive Tables Browser"
3. Click on "No-Stop Limits for Air Diving" (blue card)

5. Click "View Table"

================================================================================
WHAT IF SOMETHING GOES WRONG:
================================================================================

The widget is designed to be bulletproof, but if you have issues:

1. Try a different web browser
2. Make sure file is not read-only (for USB: disable protection)
3. Try opening from a different location
4. Check that file size is correct (~530KB)
5. Verify you're opening widget-tables-complete.html (not a text file)

If all else fails: The file is self-contained and valid. It will work on any
browser on any computer as long as the file itself is intact.
